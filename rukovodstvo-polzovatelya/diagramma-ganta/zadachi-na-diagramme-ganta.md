# Задачи на диаграмме Ганта

## Просмотр диаграммы Ганта

По умолчанию на диаграмме Ганта отображаются все задачи. Чтобы увидеть только нужные задачи, используйте фильтры:

<figure><img src="../../.gitbook/assets/image (1306).png" alt=""><figcaption></figcaption></figure>

## Настройка табличной части

Справа от диаграммы находится таблица со следующими колонками по умолчанию: ID, Тема, Дата начала, Дата окончания, Тип, Статус, Исполнитель и Приоритет.

1. **Сортировка**

По умолчанию сортировка автоматическая - по ID задачи. Чтобы изменить порядок нажмите на кнопку [**Сортировка**](../sortirovka/sortirovka-zadach.md).&#x20;

<figure><img src="../../.gitbook/assets/image (1307).png" alt=""><figcaption></figcaption></figure>

2. **Группировка**

Для изменения структуры отображения на диаграмме Ганта, нажмите на кнопку "Группировка" и выберите необходимый вариант отображения.

<figure><img src="../../.gitbook/assets/image (1311).png" alt=""><figcaption></figcaption></figure>

Доступные варианты:

* По иерархии задач - по умолчанию сортирует задачи в порядке возрастания ID, при этом подзадачи наглядно отображены рядом с родительскими задачами.&#x20;
* Без группировки - простая сортировка задач по ID без отображения подзадач;

3. **Изменение колонок**

При необходимости добавить или убрать колонки, нажмите **Колонки** в контекстном меню диаграммы Ганта.

<figure><img src="../../.gitbook/assets/image (1308).png" alt=""><figcaption></figcaption></figure>

В открывшемся окне можно настроить, какие данные показывать. Для удаления колонки нажмите на кнопку рядом с её названием.

<figure><img src="../../.gitbook/assets/image (132).png" alt="" width="375"><figcaption></figcaption></figure>

Нажмите кнопку "**Применить**", чтобы применить внесенные изменений.

Нажмите кнопку "**Сбросить**", чтобы сбросить изменения.

## Сохранение настроек

Голубая точка появится на кнопках, если вы добавили новые колонки, настроили фильтры или сортировку. Чтобы сохранить настройки, нажмите "**Сохранить вид**".

<figure><img src="../../.gitbook/assets/image (1309).png" alt=""><figcaption></figcaption></figure>

## Чтение диаграммы Ганта

Справа от таблицы задач отображается диаграмма Ганта. Каждая задача отображается горизонтальной полосой, где левая граница - дата начала, а правая граница - дата окончания задачи. Длина полосы соответствует его длительности задачи.

Задачи разных типов отображаются разными цветами, вехи (контрольные точки) отображаются в виде ромбов.

Зависимости задач показаны стрелками. Нажав на стрелку можно увидеть подробности связи.

<figure><img src="../../.gitbook/assets/image (1313).png" alt=""><figcaption></figcaption></figure>

### Навигация по временной шкале

### Масштаб

Используйте меню в верхней части диаграммы, чтобы изменить масштаб отображения: дни, недели, кварталы, года.

<figure><img src="../../.gitbook/assets/image (1314).png" alt=""><figcaption></figcaption></figure>

### Текущая дата

Красная пунктирная линия показывает текущую дату и обновляется в реальном времени. Чтобы быстро вернуться к сегодняшней дате, нажмите кнопку "**Сегодня**":

<figure><img src="../../.gitbook/assets/image (1315).png" alt=""><figcaption></figcaption></figure>

### Выбор периода&#x20;

С помощью кнопок навигации перемещайтесь по временной шкале, например, следующие 3 месяца, следующие 5 лет:

<figure><img src="../../.gitbook/assets/image (1316).png" alt=""><figcaption></figcaption></figure>

Чтобы задать конкретную начальную дату для планирования, используйте поле "Дата начала периода":

<figure><img src="../../.gitbook/assets/image (1317).png" alt=""><figcaption></figcaption></figure>

## Создание задач

Чтобы создать новую задачу, нажмите на кнопку "**+ Задача**". Она доступна в любом виде отображения: в списке задач, на доске или на диаграмме Ганта. Подробнее можно прочесть на странице "[Создание задач](../zadachi/sozdanie-zadach.md#sozdanie-zadach)".

## Изменение сроков задач

На диаграмме Ганта можно менять продолжительность задачи, двигая её границы - в этом случае изменится длительность задачи, дата ее начала и окончания. Статью про даты и продолжительность задач [можно найти тут](../zadachi/daty-i-prodolzhitelnost-zadach.md).

## Редактирование задач

Чтобы открыть задачу для полного редактирования, кликните на её ID или тему в таблице, либо на полоску на диаграмме. Подробнее можно ознакомиться [в этой статье](../zadachi/redaktirovanie-zadach.md).
