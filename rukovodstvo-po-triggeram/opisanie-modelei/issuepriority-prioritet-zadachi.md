# IssuePriority - Приоритет задачи

## Поля

| Ключ        | Тип     | Название                      | Описание |
| ----------- | ------- | ----------------------------- | -------- |
| id          | integer | ID                            |          |
| name        | string  | Название                      |          |
| is\_default | boolean | Приоритет задачи по умолчанию |          |
| active      | boolean | Активный/Архивный             |          |
