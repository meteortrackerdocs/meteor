# Интеграции

METEOR предлагает интеграцию с различными инструментами для оптимизации рабочих процессов и расширения возможностей платформы. Поддержка и обслуживание этих интеграций осуществляется командой METEOR для обеспечения совместимости и стабильности.



**Обзор раздела**:

1. Интеграция с [GitHub](integraciya-s-github.md);
2. Интеграция с [GitLab](integraciya-s-gitlab.md);
3. Интеграция с [mymeet](integraciya-s-mymeet.ai.md);
4. Интеграция с [Albato](integraciya-s-albato.md).
